El objetivo de este notebook es mostrar las primeras salidas de nuestra red. Un detalle interesante es la reconstrucción del mapa completo a través de los parches. De momento, existen dos maneras de hacerlo:

    1. A través de la información adicional: tiempo y espacio, y de esta manera, asociar cada parche a su cuadrante y registro temporal. Si bien esta manera efectivamente conduce a armar mapas completos y resulta muy útil a la hora de querer situar cualquier tipo de parche en su contexto temporal y espacial, no es una manera tan directa (aunque igual de posible) si uno quiere trabajar sobre algún conjunto en específico.
    1. Obteniendo salidas para el stack completo de datos. Entregando el stack completo y ordenado a la red nos permitirá obtener salidas ordenadas inmediatamente y, como utilizamos ciclos for para el cortado de parches, fácilmente podemos reconstruir cualquier mapa si solo tomamos múltiplos de 156 parches de nuestro stack de salidas. Además, como cada conjunto utilizado para el entrenamiento de la red fue previamente guardado, podemos crear una función que simplemente compare cada parche del stack completo (entrada de la red) con los parches contenidos para cada conjunto y saber a qué conjunto pertenece dicho parche, de esta manera tendremos una etiqueta asociada a cada parche ordenado que nos indique a qué conjunto pertenece y cuál es su correspondiente salida de manera inmediata.

Siguiendo el procedimiento (2) es que se creó la variable idx_ordenque contiene la etiqueta: {tr: 0, va: 1, te: 2} asociada a cada parche del stack completo.

Desnormalización

En el fichero anterior cada una de las variables fueron normalizadas en intervalos del \([0,1]\). Utilizaremos los factores de normalización creados durante este proceso para desnormalizar el stack completo para las variables: input, target y output_ord

Code 
def normalization(data_in=input,inverse=False, scale_factor=[1,1]):
    '''Método que normaliza los valores entregados.
        input:
            data_in: [tensor] Tensor con los parches de la variable que forma parte del input de la Red
            inverse: [Boolean, default=False] True en caso de devolver la transformación
        return: 
            variable normalizada
    '''
    if not inverse:
        scale_factor_ = np.max(np.abs(data_in))
        scale_factor_2 = np.min(np.abs(data_in))
        data_out = (data_in-scale_factor_2)/(scale_factor_ - scale_factor_2)
        scale_factor_ = [scale_factor_,scale_factor_2]
    else:
        data_out=(data_in * (scale_factor[0]-scale_factor[1]))+scale_factor[1]
        scale_factor_ = scale_factor
    return data_out, scale_factor_

Como ya explicamos al comienzo del fichero, utilizaremos el orden en los bloques de cada variable para la reconstrucción del mapa completo. Cada 156 parches, iremos ubicando cada parche en su posición respectiva, comenzando por arriba a la izquierda y luego iremos agregando parches hasta completar la fila para luego pasar a la siguiente. Esto lo hace la función mapa_restaurado, la cual se encarga de ir situando los parches ordenados en esta matriz de (384, 416) que corresponde a las dimensiones del mapa completo.

def mapa_restaurado(tensor_pic):

    posiciones=[]
    for i in range (12):
        for j in range(13):
            posiciones.append( (32*i, 32*j))

    image_size = 32
    imagen_unida = np.zeros((12 * image_size, 13* image_size), dtype=np.float32)

    for i,j in enumerate( range(156)):
        tensor_data = tensor_pic[j].numpy()  
        imagen_unida[ posiciones[i][0]:posiciones[i][0] + image_size,  posiciones[i][1]:posiciones[i][1] + image_size ] = tensor_data   

    return imagen_unida

La función determinar_inputs se encarga de unir los 156 parches correspondientes a un registro temporal \(i\) y entregar las salidas: inp, target, predict, residual, en formato de mapa completo con dimensiones (384,416)

def determinar_inputs(i=int, inp=input_stack, target=observed_stack, predict= fore_stack):
    inp= mapa_restaurado(input_stack[(156*(i-1)):(156*(i))])
    target= mapa_restaurado(observed_stack[(156*(i-1)):(156*(i))])
    predict= mapa_restaurado( fore_stack[(156*(i-1)):(156*(i))])
    residual=  mapa_restaurado(     observed_stack[(156*(i-1)):(156*(i))] - fore_stack[(156*(i-1)):(156*(i))])
    #labels= combined_image(etiquetas[(12*(i-1)):(12*(i))])

    return inp, target, predict, residual
